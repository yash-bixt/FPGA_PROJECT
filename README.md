# FPGA Digital Storage Oscilloscope (DSO)

## Overview
This project implements a basic DSO using Nexys 4 FPGA with VGA output.

## Features
- Real-time waveform display
- VGA interface (640x480)
- ADC input using XADC
- Arduino-based signal input

## Hardware Used
- Nexys 4 (Artix-7)
- Arduino Nano (signal source)

## How to Run
1. Open project in Vivado
2. Generate bitstream
3. Program FPGA
4. Connect VGA monitor
